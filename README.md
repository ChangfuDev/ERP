# ERP
项目使用的是SSH框架，业务流程为：采购订单申请—>订单审核—>采购运输—>入库。 <br/> <br/>
模块主要分为：基础维护模块（员工模块，商品模块，供应商模块，仓库模块，菜单模块，订单模块）、采购业务、审核业务、运输业务、入库业务 <br/> <br/>
实现的功能：业务主线流程（采购订单申请—>订单审核—>采购运输—>入库）实现<br/>
            权限系统的实现 <br/> <br/>
       <br/>
